# Influenza Surveillance Dashboard

An interactive dashboard displaying influenza data from the UK Health Security Agency (UKHSA).

## Features
- ICU/HDU admission rates over time
- Testing positivity by age group
- Live data refresh from UKHSA API
- Interactive visualizations

## Running the Dashboard

### Locally with Voila:
```bash
voila InfluenzaDashboard.ipynb
```

## Data Source
UK Health Security Agency (UKHSA): https://ukhsa-dashboard.data.gov.uk/
